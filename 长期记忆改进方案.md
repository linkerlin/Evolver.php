# Evolver.php 长期记忆改进方案

> 基于 memory-lancedb-pro (OpenClaw 生产级记忆插件) 的对比分析

## 1. 两项目对比分析

### 1.1 架构对比

| 维度 | memory-lancedb-pro | Evolver.php |
|------|-------------------|-------------|
| **存储引擎** | LanceDB (向量数据库) | SQLite (关系型) |
| **检索方式** | 向量 + BM25 混合检索 | 无检索系统 (仅 JSONL 追加) |
| **嵌入支持** | 完整的 Embedding 抽象层 | 无 |
| **衰减模型** | Weibull 拉伸指数衰减 | 简单指数半衰衰减 |
| **层级管理** | Core/Working/Peripheral 三层 | 无层级概念 |
| **提取系统** | LLM 驱动 6 类别智能提取 | 基于规则的信号提取 |
| **元数据** | Smart Metadata (L0/L1/L2 + tier + confidence) | 简单 JSON 序列化 |
| **去重策略** | 向量预过滤 + LLM 语义决策 | Jaccard 相似度 |

**三层晋升系统**:
- **Core** (decay floor 0.9): 身份级事实，几乎永不忘

- **Peripheral** (decay floor 0.5): 低优先级或老化记忆


**多层评分管线**:
1. **RRF 融合**: 向量相似度 + BM25 关键词
2. **跨编码器重排**: Jina/Voyage/Pinecone Rerank API
3. **生命周期衰减加权**: Weibull 新鲜度 + 访问频率
4. **长度归一化**: 防止长条目霸占结果
5. **硬最低分过滤**: 低于阈值直接丢弃
6. **MMR 多样性去重**: 相似度过高的条目降级

#### 2.2.1 实现 Weibull 衰减引擎

```php
// src/DecayEngine.php
final class DecayEngine
{
    private const MS_PER_DAY = 86400000;

    public function __construct(
        private float $recencyHalfLifeDays = 30.0,
        private float $recencyWeight = 0.4,
        private float $frequencyWeight = 0.3,
        private float $intrinsicWeight = 0.3,
        private float $betaCore = 0.8,
        private float $betaWorking = 1.0,
        private float $betaPeripheral = 1.3,
    ) {}

    /**
     * 计算复合衰减分数
     * composite = rw * recency + fw * frequency + iw * intrinsic
     */
    public function score(DecayableMemory $memory, int $now = null): DecayScore
    {
        $now = $now ?? time() * 1000;

        $recency = $this->computeRecency($memory, $now);
        $frequency = $this->computeFrequency($memory);
        $intrinsic = $this->computeIntrinsic($memory);

        $composite = ($this->recencyWeight * $recency)
                   + ($this->frequencyWeight * $frequency)
                   + ($this->intrinsicWeight * $intrinsic);

        return new DecayScore(
            memoryId: $memory->id,
            recency: $recency,
            frequency: $frequency,
            intrinsic: $intrinsic,
            composite: $composite,
        );
    }

    /**
     * Weibull 拉伸指数衰减
     * recency = exp(-λ * daysSince^β)
     * λ = ln(2) / effectiveHalfLife
     * effectiveHalfLife = baseHalfLife * exp(μ * importance)
     */
    private function computeRecency(DecayableMemory $m, int $now): float
    {
        $lastActive = $m->accessCount > 0 ? $m->lastAccessedAt : $m->createdAt;
        $daysSince = max(0, ($now - $lastActive) / self::MS_PER_DAY);

        // Importance 调制半衰期
        $mu = 1.5; // 调制系数
        $effectiveHL = $this->recencyHalfLifeDays * exp($mu * $m->importance);
        $lambda = M_LN2 / $effectiveHL;

        // Tier-specific beta
        $beta = match ($m->tier) {
            'core' => $this->betaCore,
            'peripheral' => $this->betaPeripheral,
            default => $this->betaWorking,
        };

        return exp(-$lambda * pow($daysSince, $beta));
    }

    /**
     * 对数饱和频率曲线
     * base = 1 - exp(-accessCount / 5)
     * 带时间加权访问模式奖励
     */
    private function computeFrequency(DecayableMemory $m): float
    {
        $base = 1 - exp(-$m->accessCount / 5);
        if ($m->accessCount <= 1) return $base;

        $accessSpanDays = max(1, ($m->lastAccessedAt - $m->createdAt) / self::MS_PER_DAY);
        $avgGapDays = $accessSpanDays / max($m->accessCount - 1, 1);
        $recentnessBonus = exp(-$avgGapDays / 30);

        return $base * (0.5 + 0.5 * $recentnessBonus);
    }

    /**
     * 内在价值 = importance × confidence
     */
    private function computeIntrinsic(DecayableMemory $m): float
    {
        return $m->importance * $m->confidence;
    }
}
```

#### 2.2.2 实现三层晋升管理器

```php
// src/TierManager.php
final class TierManager
{
    public function __construct(
        private int $coreAccessThreshold = 10,
        private float $coreCompositeThreshold = 0.7,
        private float $coreImportanceThreshold = 0.8,
        private float $peripheralCompositeThreshold = 0.15,
        private int $peripheralAgeDays = 60,
        private int $workingAccessThreshold = 3,
        private float $workingCompositeThreshold = 0.4,
    ) {}

    /**
     * 评估记忆是否应该改变层级
     */
    public function evaluate(TierableMemory $memory, DecayScore $score, int $now = null): ?TierTransition
    {
        $now = $now ?? time() * 1000;
        $ageDays = ($now - $memory->createdAt) / 86400000;

        return match ($memory->tier) {
            'peripheral' => $this->tryPromoteToWorking($memory, $score),
            'working' => $this->tryPromoteToCore($memory, $score)
                      ?? $this->tryDemoteToPeripheral($memory, $score, $ageDays),
            'core' => $this->tryDemoteToWorking($memory, $score),
            default => null,
        };
    }

    private function tryPromoteToWorking(TierableMemory $m, DecayScore $s): ?TierTransition
    {
        if ($m->accessCount >= $this->workingAccessThreshold
            && $s->composite >= $this->workingCompositeThreshold) {
            return new TierTransition(
                memoryId: $m->id,
                fromTier: 'peripheral',
                toTier: 'working',
                reason: "Access ({$m->accessCount}) >= {$this->workingAccessThreshold}, composite ({$s->composite}) >= {$this->workingCompositeThreshold}",
            );
        }
        return null;
    }

    private function tryPromoteToCore(TierableMemory $m, DecayScore $s): ?TierTransition
    {
        if ($m->accessCount >= $this->coreAccessThreshold
            && $s->composite >= $this->coreCompositeThreshold
            && $m->importance >= $this->coreImportanceThreshold) {
            return new TierTransition(
                memoryId: $m->id,
                fromTier: 'working',
                toTier: 'core',
                reason: "High access ({$m->accessCount}), composite ({$s->composite}), importance ({$m->importance})",
            );
        }
        return null;
    }

    // ... 降级逻辑
}
```

```php
final class OpenAIEmbedder implements EmbedderInterface

    {

        curl_setopt_array($ch, [
            CURLOPT_POST => true,
            CURLOPT_RETURNTRANSFER => true,

                'input' => $text,
                'dimensions' => $this->dimensions,
            ]),
        ]);

        curl_close($ch);

        $data = json_decode($response, true);

    }
```

```php
// 由于 SQLite 不原生支持向量，使用 JSON 存储向量 + 内存索引

// src/VectorStore.php
final class VectorStore
{
    private array $vectorCache = []; // 内存向量缓存
    private EmbedderInterface $embedder;

    public function __construct(
        private Database $db,
        EmbedderInterface $embedder = null,
    ) {
        $this->embedder = $embedder ?? new NullEmbedder();
    }

    /**
     * 存储带向量的记忆
     */
    public function store(string $type, string $id, string $text, array $meta = []): void
    {
        $vector = $this->embedder->embed($text);
        $vectorJson = $vector ? json_encode($vector) : null;

        $this->db->exec(
            "INSERT OR REPLACE INTO vector_index (id, type, text, vector, metadata) VALUES (?, ?, ?, ?, ?)",
            [$id, $type, $text, $vectorJson, json_encode($meta)]
        );

        if ($vector) {
            $this->vectorCache[$id] = $vector;
        }
    }

    /**
     * 余弦相似度搜索
     */
    public function search(string $query, int $limit = 10, float $minScore = 0.3): array
    {
        $queryVector = $this->embedder->embed($query);
        if (!$queryVector) return [];

        // 加载所有向量到内存 (小数据集可接受)
        $this->loadVectorsIfNeeded();

        $scores = [];
        foreach ($this->vectorCache as $id => $vector) {
            $similarity = $this->cosineSimilarity($queryVector, $vector);
            if ($similarity >= $minScore) {
                $scores[$id] = $similarity;
            }
        }

        arsort($scores);
        return array_slice($scores, 0, $limit, true);
    }

    private function cosineSimilarity(array $a, array $b): float
    {
        $dot = 0.0; $normA = 0.0; $normB = 0.0;
        for ($i = 0; $i < count($a); $i++) {
            $dot += $a[$i] * $b[$i];
            $normA += $a[$i] * $a[$i];
            $normB += $b[$i] * $b[$i];
        }
        $norm = sqrt($normA) * sqrt($normB);
        return $norm > 0 ? $dot / $norm : 0.0;
    }
}
```

#### 2.4.2 智能提取器

```php
// src/SmartExtractor.php
final class SmartExtractor
{
    private const MAX_MEMORIES_PER_EXTRACTION = 5;
    private const SIMILARITY_THRESHOLD = 0.7;

    public function __construct(
        private VectorStore $store,
        private EmbedderInterface $embedder,
        private LlmClient $llm,
    ) {}

    /**
     * 从对话文本提取记忆并持久化
     */
    public function extractAndPersist(string $conversationText, string $sessionKey = 'unknown'): ExtractionStats
    {
        $stats = new ExtractionStats();

        // Step 1: LLM 提取候选记忆
        $candidates = $this->extractCandidates($conversationText);
        if (empty($candidates)) {
            return $stats;
        }

        // Step 2: 处理每个候选 (去重 → 合并/创建 → 存储)
        foreach (array_slice($candidates, 0, self::MAX_MEMORIES_PER_EXTRACTION) as $candidate) {
            try {
                $this->processCandidate($candidate, $sessionKey, $stats);
            } catch (\Throwable $e) {
                error_log("SmartExtractor: failed to process candidate: {$e->getMessage()}");
            }
        }

        return $stats;
    }

    /**
     * LLM 提取候选记忆 (L0/L1/L2 结构)
     */
    private function extractCandidates(string $text): array
    {
        $prompt = $this->buildExtractionPrompt($text);
        $result = $this->llm->completeJson($prompt);

        if (!isset($result['memories']) || !is_array($result['memories'])) {
            return [];
        }

        $candidates = [];
        foreach ($result['memories'] as $raw) {
            $category = MemoryCategory::tryFrom(strtolower($raw['category'] ?? ''));
            if (!$category) continue;

            $abstract = trim($raw['abstract'] ?? '');
            if (strlen($abstract) < 5) continue;

            $candidates[] = new CandidateMemory(
                category: $category,
                abstract: $abstract,
                overview: trim($raw['overview'] ?? ''),
                content: trim($raw['content'] ?? ''),
            );
        }

        return $candidates;
    }

    /**
     * 两阶段去重: 向量预过滤 + LLM 语义决策
     */
    private function deduplicate(CandidateMemory $candidate, array $vector): DedupResult
    {
        // Stage 1: 向量相似度搜索
        $similar = $this->store->search($candidate->abstract, 5, self::SIMILARITY_THRESHOLD);

        if (empty($similar)) {
            return new DedupResult(decision: 'create', reason: 'No similar memories found');
        }

        // Stage 2: LLM 语义决策
        return $this->llmDedupDecision($candidate, $similar);
    }

    private function processCandidate(CandidateMemory $candidate, string $sessionKey, ExtractionStats $stats): void
    {
        // Profile 类别始终合并
        if ($candidate->category->alwaysMerge()) {
            $this->handleProfileMerge($candidate, $sessionKey);
            $stats->merged++;
            return;
        }

        // 生成嵌入向量
        $embeddingText = "{$candidate->abstract} {$candidate->content}";
        $vector = $this->embedder->embed($embeddingText);

        if (!$vector) {
            $this->storeCandidate($candidate, [], $sessionKey);
            $stats->created++;
            return;
        }

        // 去重决策
        $dedupResult = $this->deduplicate($candidate, $vector);

        match ($dedupResult->decision) {
            'create' => (function() use ($candidate, $vector, $sessionKey, $stats) {
                $this->storeCandidate($candidate, $vector, $sessionKey);
                $stats->created++;
            })(),
            'merge' => (function() use ($candidate, $dedupResult, $stats) {
                if ($candidate->category->supportsMerge() && $dedupResult->matchId) {
                    $this->handleMerge($candidate, $dedupResult->matchId);
                    $stats->merged++;
                } else {
                    $stats->created++;
                }
            })(),
            'skip' => $stats->skipped++,
            'supersede' => (function() use ($candidate, $vector, $dedupResult, $sessionKey, $stats) {
                if ($candidate->category->isTemporalVersioned() && $dedupResult->matchId) {
                    $this->handleSupersede($candidate, $vector, $dedupResult->matchId, $sessionKey);
                    $stats->created++;
                    $stats->superseded++;
                }
            })(),
            default => $stats->created++,
        };
    }

    // ... handleProfileMerge, handleMerge, handleSupersede 等方法
}
```

### 2.5 Phase 5: 混合检索系统 (优先级: 中)

#### 2.5.1 检索器实现

```php
// src/MemoryRetriever.php
final class MemoryRetriever
{
    public function __construct(
        private VectorStore $vectorStore,
        private Database $db,
        private DecayEngine $decayEngine,
        private RetrievalConfig $config = new RetrievalConfig(),
    ) {}

    /**
     * 混合检索: 向量 + BM25 + Rerank + 生命周期衰减
     */
    public function retrieve(RetrievalContext $context): array
    {
        // 1. 向量搜索
        $vectorResults = $this->vectorStore->search(
            $context->query,
            $this->config->candidatePoolSize,
            0.1
        );

        // 2. BM25 全文搜索 (SQLite FTS5)
        $bm25Results = $this->bm25Search($context);

        // 3. RRF 融合
        $fused = $this->fuseResults($vectorResults, $bm25Results);

        // 4. 应用生命周期衰减
        $lifecycleRanked = $this->applyDecayBoost($fused);

        // 5. 长度归一化
        $normalized = $this->applyLengthNormalization($lifecycleRanked);

        // 6. 硬最低分过滤
        $filtered = array_filter($normalized, fn($r) => $r['score'] >= $this->config->hardMinScore);

        // 7. MMR 多样性去重
        $deduplicated = $this->applyMMRDiversity($filtered);

        return array_slice($deduplicated, 0, $context->limit);
    }

    /**
     * SQLite FTS5 BM25 搜索
     */
    private function bm25Search(RetrievalContext $context): array
    {
        // 使用 SQLite FTS5 扩展 (如果可用)
        $sql = "SELECT id, text, bm25(memory_fts) as score
                FROM memory_fts
                WHERE memory_fts MATCH ?
                ORDER BY score DESC
                LIMIT ?";

        try {
            $rows = $this->db->fetchAll($sql, [$context->query, $this->config->candidatePoolSize]);
            return array_map(fn($r) => ['id' => $r['id'], 'score' => 1 / (1 + exp(-$r['score'] / 5))], $rows);
        } catch (\Throwable $e) {
            // FTS5 不可用，回退到 LIKE 搜索
            return $this->lexicalFallbackSearch($context);
        }
    }

    /**
     * RRF 融合
     */
    private function fuseResults(array $vectorResults, array $bm25Results): array
    {
        $allIds = array_unique(array_merge(
            array_keys($vectorResults),
            array_keys($bm25Results)
        ));

        $fused = [];
        foreach ($allIds as $id) {
            $vectorScore = $vectorResults[$id] ?? 0;
            $bm25Score = $bm25Results[$id] ?? 0;

            // 加权融合
            $fusedScore = ($vectorScore * $this->config->vectorWeight)
                        + ($bm25Score * $this->config->bm25Weight);

            // BM25 高分保底 (精确关键词匹配)
            if ($bm25Score >= 0.75) {
                $fusedScore = max($fusedScore, $bm25Score * 0.92);
            }

            $fused[$id] = $fusedScore;
        }

        arsort($fused);
        return $fused;
    }

    /**
     * 应用生命周期衰减加权
     */
    private function applyDecayBoost(array $results): array
    {
        $boosted = [];
        foreach ($results as $id => $score) {
            // 从数据库加载记忆元数据
            $memory = $this->loadMemoryMetadata($id);
            if (!$memory) {
                $boosted[$id] = $score;
                continue;
            }

            $decayScore = $this->decayEngine->score($memory);
            $tierFloor = $this->getTierFloor($memory->tier);

            // 乘数: minScore + (1 - minScore) * max(tierFloor, composite)
            $multiplier = $this->config->searchBoostMin
                        + ((1 - $this->config->searchBoostMin) * max($tierFloor, $decayScore->composite));

            $boosted[$id] = $score * min(1, max($this->config->searchBoostMin, $multiplier));
        }

        arsort($boosted);
        return $boosted;
    }

    /**
     * MMR 多样性去重
     */
    private function applyMMRDiversity(array $results, float $threshold = 0.85): array
    {
        if (count($results) <= 1) return $results;

        $selected = [];
        $deferred = [];

        foreach ($results as $id => $score) {
            $tooSimilar = false;
            foreach (array_keys($selected) as $selectedId) {
                $similarity = $this->vectorStore->similarity($id, $selectedId);
                if ($similarity > $threshold) {
                    $tooSimilar = true;
                    break;
                }
            }

            if ($tooSimilar) {
                $deferred[$id] = $score;
            } else {
                $selected[$id] = $score;
            }
        }

        return array_merge($selected, $deferred);
    }
}
```

## 5. 总结

本改进方案从 memory-lancedb-pro 中提取了以下核心设计理念:

1. **结构化元数据**: L0/L1/L2 三层内容层级，支持高效检索和深度分析
2. **生命周期管理**: Weibull 衰减 + 三层晋升系统，让重要记忆浮上来
3. **智能提取**: LLM 驱动的 6 类别分类 + 语义去重
4. **混合检索**: 向量 + BM25 + Rerank + MMR 多样性

通过渐进式实施，Evolver.php 将获得生产级长期记忆能力，同时保持轻量部署和私有化存储的核心优势。
